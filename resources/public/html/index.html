MMLife

FieldMinimap

MMLife это многопользовательская реализация Conway's Game of Life . Все пользователи имеют доступ к единому полю, замкнутому в тор. В каждый момент времени можно наблюдать только часть поля сквозь небольшое "окошко", причем миникарта (minimap) позволяет перемещать это "окошко" в нужное место поля.

Каждый пользователь может "поселять" в видимые ячейки "организмы", причем цвет ячеек будет соответствовать цвету пользователя. Для этого нужно, кликая по полю (field), выбрать ячейки для заселения и нажать кнопку "заселить!".
